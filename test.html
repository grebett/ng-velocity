AngularJS directive for velocity.js custom animation.

For Velocity UI integration, check one of these two libraries :
https://github.com/cgwyllie/angular-velocity
https://github.com/rosslavery/velocity-ui-angular

Animation and chained animation based on id

#mydiv

Click on one div will trigger velocity attribute effect, after data-delay ms.
If data-follow is provided, trigger next the target element(s) animation with optional data-follow-delay overwriting initial animation delay.

Code:
<div id="mydiv" velocity="effect1" data-delay="500" data-follow="myotherdiv" data-follow-delay="2000">

#myotherdiv

Code:
<iv id="myotherdiv" velocity="effect1" data-follow="mydiv" data-follow-delay="100">

Animation and chained animation based on class

#myotherdiv

Click on this div will animate it and trigger animation for the two .classy divs below.

Code:
<id="mythirddiv" velocity="effect1" data-follow=".classy" data-follow-delay="1000">

.classy

Code:
<div class="classy" velocity="effect1">
